ℹ️ Assumptions & Notes

Ground-Mount Layout: This tool is designed for ground-mounted solar arrays with adjustable azimuth (compass direction). The azimuth setting (0-360 degrees) is for site planning documentation and does not affect BOM calculations.

Footprint Calculation: Estimated footprint (sq ft) is calculated as: (max row width in ft) x (total array depth in ft, including row gaps and spacing).

Waste Factor: Applied uniformly to all clamps, brackets, wire management, and splices. Modules and inverters are not subject to waste.

Section Types: 8-Panel sections use 2 rows x 4 panels/row. 10-Panel sections use 2 rows x 5 panels/row. Structural and electrical components scale with section type.

Data Persistence: Project state is automatically saved to browser local storage. Restore the last saved project using the tools button below.
